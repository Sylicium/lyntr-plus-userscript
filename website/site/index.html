How to get Lyntr+

It's super simple, you only need to have Chrome or Firefox with the TamperMonkey extension or equivalent !

1

Download TamperMonkey

This browser extension allows to add userscripts on certain webpages to edit the content. Lyntr+ is using this in order to work fine.
Download TamperMonkey for Chrome or Firefox or the open-sourced equivalent ViolentMonkey for Chrome or Firefox
2

Install the script

Click here to download the latest version of Lyntr+, or browse the GitHub releases to install a specific version.
3

It's Done !

Just reload the page, and you're ready to go !
Bringing features
& removing issues

Our goal is to make the user experience the best possible, by adding features before their official release on vanilla Lyntr, or quite simply which will never be released. 

In addition to the many added features, Lyntr+ fixes most of the bugs not fixed in the vanilla version, while maintaining stability in use.
Bugs such as:
- Too long usernames hiding the 3 dots
- Feed button not scrolling back to top on page change
- Unclickable @mentions and links
And much more ! To submit new bugs from Lyntr to fix in Lyntr+, please open an issue on the GitHub, or let us know on the Discord !

Some features

Clickable links and transparency

Tired of not being able to click that link in a lynt ? Its now possible with Lyntr+
This feature is also parsing in the biographies, so feel free to put your links !

With the quick settings, you can change the transparency for the lynts so you can better see your beautiful image background !

Custom background and colors

You want to change the background color ? Even better, put an image there ? Well now you can !

Icons in your notification tab also have more contrast with added colors.

Tiny yes, but super useful.

- quickly reload the page on cloudflare error,
- jump to the top of the feed when changing feed
